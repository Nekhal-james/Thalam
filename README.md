> **Thalam** (Sanskrit for *Rhythm*) is a dynamic cultural intelligence system designed to map, interpret, and visualize the intricate interconnectedness of Kerala’s living traditions, rituals, and artistic lineages.

---

## ✨ Core Experiences

### 📍 The Pulse Map
A geospatial archive of Kerala’s cultural landscape. From the trance-inducing rituals of **Theyyam** in the North to the majestic temple pageantry of **Pooram** in the center, explore the geographic soul of the land through an interactive, dark-themed cartographic interface.

### 🧬 Cultural DNA Explorer
Go beneath the surface. Powered by **vis-network**, this graph visualization tracks the "DNA" of traditions—showing how Koodiyattam influenced Kathakali, or how ancient trade routes with the Arab world birthed the Malabar Biryani. Discover shared roots and thematic parallels across time and space.

### 📉 Trend Harmonics
A data-driven lens on preservation and vitality. Using **Recharts**, we visualize the "preservation health" and emotional resonance of diverse cultural categories, helping researchers identify traditions at risk and those flourishing in the modern era.

### 🧘 Guru AI Insight
Integration with **Google Gemini (3-flash-preview)** provides scholarly, narrative interpretations of every tradition. *Guru* doesn't just provide facts; it analyzes structural anthropology, explaining the "why" behind the ritual.

## 🛠️ Technical Fabric

Thalam is built with a focus on fluidity, cinematic UI, and deep data relationships.

- **Frontend:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) for high-performance ESM-based development.
- **Motion:** [Framer Motion](https://www.framer.com/motion/) for immersive transitions and micro-interactions.
- **Maps:** [React Leaflet](https://react-leaflet.js.org/) for a custom charcoal-themed geospatial UI.
- **Graph Logic:** [Vis-Network](https://visjs.github.io/vis-network/docs/network/) for real-time relationship Mapping.
- **Intelligence:** [@google/genai](https://www.npmjs.com/package/@google/genai) for context-aware cultural analysis.
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) with a custom typography palette (Playfair Display & Inter).

## 🚀 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- A **Gemini API Key** from [Google AI Studio](https://aistudio.google.com/)

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/nekhaljames/thalam.git

# Install dependencies
npm install

# Configure environment
# Add your API key to a .env file:
# GEMINI_API_KEY=your_key_here
```

### 3. Launch
```bash
npm run dev
```

## 🏛️ Architecture of the Soul

The project organises cultural data into a "Relationship Matrix":

- **Entities:** Rituals, Art Forms, Historical Persons, Festivals, and Objects.
- **Relationships:** `influenced_by`, `derivative_of`, `shares_root_with`, `geographically_linked`.
- **Metrics:** `preservation_score` and `emotional_resonance`.
